
# Etch-a-Sketch

A browser-based Etch-a-Sketch drawing app built as part of The Odin Project Foundations curriculum.

## Features

- Generates a dynamic grid using JavaScript DOM manipulation
- Allows users to choose the number of squares per side
- Resizes the grid while keeping the same canvas size
- Hover over squares to draw
- Generates random RGB colors for each interaction
- Includes input validation to prevent oversized grids

## Technologies Used

- HTML
- CSS
- JavaScript
- Flexbox

## What I Learned

Through this project I practiced:

- Creating and modifying DOM elements with JavaScript
- Using loops to generate elements dynamically
- Adding event listeners
- Working with functions and parameters
- Manipulating element styles with JavaScript
- Handling user input and validation
- Using Flexbox to create responsive layouts
